## Error 201

### Description:
The application failed to generate required files due to a permissions error.


### Type:
General/Permissions

### Solutions:

#### Linux / MacOS
- Ensure that you are running the program as a super user (use `sudo`)

#### Windows
- Ensure that you are running the program as an administrator
